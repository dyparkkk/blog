# IDE 단축키(VSCode, IntelliJ)

하나씩 계속 추가 할 예정이다.  
*윈도우 기준으로 작성*

## VSCode

- 하단에 터미널 띄우기  
  `cnrl` + `` ` `` 

- 자동 정렬  
  `cnrl + k + f` (순서대로)

마크다운
- 실시간 미리보기  
  `ctrl + K V`

- 미리보기 페이지  
  `ctrl + shift + V`

## IntelliJ
- 자동 정렬  
  `ctrl + alt + l`
    


- 메소드 추출  
  `ctrl + alt + m`

- 인라인 (라인 합치기)  
  `ctrl + alt + n`

- 멀티커서(multi-curser)  
  `ctrl * 2` + 방향키  
  `shift + alt` + 마우스커서  

- 라인복사(Duplicate Line)  
  `ctrl + d`

- 탭 끄기  
  `ctrl + F4`
